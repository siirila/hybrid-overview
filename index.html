Choosing a framework

Good Idea - Select a framework based on your existing skills and experience

Bad Idea - Let a business executive tell you what framework to use

Framework considerations:

  • Mobile support
  • Performance
  • Ease of use
  • Community
  • Maturity

If you are new to modern web development, Single Page Apps, or MVC use an opinionated framework

Good opinionated frameworks include:

  • AngularJS
  • Ionic - Mobile specific, built on AngularJS
  • Ember

If you have some experience with building Single Page Apps, consider building your own structure with best of components

  • Backbone.js - ???
